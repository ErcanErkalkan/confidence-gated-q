# Reproducibility Artifact

The ASOC submission artifact is prepared as version `1.4.0`. The source repository is available at https://github.com/ErcanErkalkan/confidence-gated-q and the persistent Zenodo concept DOI is https://doi.org/10.5281/zenodo.20578927.

The artifact contains source code, JSON configurations, raw seed-level outputs, generated tables and figures, package-version records, tests, provenance files, and audit scripts for reproducing the reported analyses.

Quick verification:

```text
python scripts/reproduce_all.py --quick
```
